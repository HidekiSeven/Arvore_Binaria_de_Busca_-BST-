Implemente uma Árvore Binária de Busca (BST). 

Ele usará a sequência de dígitos do seu RA Ex: 2, 4, 0, 3, 7, 9, 3 para construir a árvore e, em seguida, exibirá os elementos em ordem crescente 
(percurso Em Ordem).

Você precisará de duas classes: No e ArvoreBinariaDeBusca.

1. Classe No (Node)
Esta classe define a estrutura básica de cada elemento na sua árvore.
2. Classe ArvoreBinariaDeBusca (BST)
Esta classe contém a lógica para inserir os nós e percorrer a árvore.
